NSArray-firstObject
===================

NSString category for MD5 hashing

Usage instructions:

1. Drag and drop NSString+MD5.h and NSString+MD5.m into your project.

2. In your view controller ```objective-c #import NSString+MD5.h;```

3. In code use:
```objective-c
    NSString *someHashedString = [@"HASH ME!" MD5];
```

That's it!
